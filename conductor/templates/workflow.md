# Workflow Template

The full, active workflow ships at `conductor/workflow.md`. This `templates/`
directory exists for the `conductor-setup` skill, which copies language-specific
code style guides from `conductor/templates/code_styleguides/` into
`conductor/code_styleguides/` during project setup.

To add a language-specific style guide, create a file at:
`conductor/templates/code_styleguides/<language>.md`
